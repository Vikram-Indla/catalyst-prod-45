> No formula is final until approved. Each must disclose numerator, denominator, included/excluded
> records, date filter, scope filter, status filter, data source, known limitation, approval status.

## Format
```
F-### | <metric>
Numerator:      <count of ...>
Denominator:    <count of ...>
Included:       <records>          Excluded: <records>
Date filter:    <which date, range>
Scope filter:   <project/release/sprint/...>
Status filter:  <which statuses count>
Data source:    <tables/views>
Limitation:     <known caveat>
Approval:       PENDING / APPROVED (decision id)
```
