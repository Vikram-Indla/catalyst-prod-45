---
name: model-diversity-review
description: Use a second model to find correlated blind spots without allowing it to change source-of-truth status.
---

1. Provide the same packet, repository snapshot, and evidence to the second model.
2. Ask for contradiction, missing-risk, architecture, and test critiques.
3. Treat every finding as unverified.
4. A native Codex verification agent must reproduce or disprove each finding.
5. Only reproduced findings enter requirements or code.
