# Discovery verification plan

## Static checks

- repository search;
- route graph;
- import/dependency graph;
- schema and migration review;
- permission-policy review;
- test inventory.

## Runtime checks

- launch local application;
- authenticate with available test role(s);
- navigate all Folio entry points;
- capture browser console and network errors;
- capture screenshots;
- exercise one create/read/update path if supported;
- do not delete or mutate production data.
